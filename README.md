# RG350 Device Layout

Pixel-perfect approximation of the RG350 device layout.

All front, top, and bottom elements of the device's layout are labeled.

Includes individual transparent PNG layer images, and the Paint.NET project file.

### FRONT
* SELECT
* START
* L3
* R3
* DPAD UP
* DPAD LEFT
* DPAD RIGHT
* DPAD DOWN
* X
* A
* Y
* B
* LED INDICATOR
* LEFT STICK
* RIGHT STICK

### TOP
* L1
* L2
* USB1
* HDMI
* HEADPHONES/AV
* USB2
* R2
* R1

### BOTTOM
* SPEAKER LEFT
* POWER
* TF
* VOLUME-
* VOLUME+
* RESET
* SPEAKER RIGHT
